Code is found in public Github repo located at https://github.com/solb0039/CS7641/tree/master/Assignment3.

Datasets were taken from UC Irving Dataset repository found at: https://archive.ics.uci.edu/ml/datasets.html
Specific dataset links for the Adult and Credit dataset are:
Adult: https://archive.ics.uci.edu/ml/datasets/Census+Income
Credit: https://archive.ics.uci.edu/ml/datasets/default+of+credit+card+clients

Important files in repo are:

main.py: High level code that imports datasets and calls separate methods for each part of assignment
credit_data_analysis.py & adult_data_analysis.py: Python functions for running each part of the assignment.  
Graphs.R:  R program to create writeup plots using ggplot.
Requirements.txt:  Python packages that need to be installed to properly run code using Python 2.7 Virtual Environment.  

Note that run time for running entire code base was >1 day on MacBook Pro laptop with 2.3Ghz processor.

Python code was adopted from jontay and iokast github repos.
